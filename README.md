# gitc : Git-curses

## Dependencies

### For Debian-based distros

#### Build dependencies
```
libncurses5-dev
libgit2-dev
```

And their equivalents for other distros.

### Install

At the moment the only way to use is to build from sources:

```
git clone https://github.com/canopeerus/gitc
cd gitc
make
sudo make install
```
